# 🔐 Desafio - Automação de Testes

## 📋 Descrição
Função de login com testes automatizados usando Mocha e Chai.

## 🎯 Funcionalidades
- ✅ Login com email e senha
- ✅ Validação de credenciais expiradas
- ✅ Tratamento de usuário não encontrado
- ✅ Verificação de senha incorreta

## 🛠️ Tecnologias
- Node.js
- Mocha
- Chai

## 🚀 Como executar

### Pré-requisitos
- Node.js instalado

### Passos

1. Clone o repositório
```bash
git clone https://github.com/JulianaQA/Desafio-Login-Testes.git
cd Desafio-Login-Testes
2. Instale as dependências
```bash
npm install
3. Execute os testes
```bash
npm test
